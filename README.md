## Kevin_zzx_CPP

This is a personal cpp library for practising various planning and control algorithms for autonomous driving.

### get start


#### dependencies

```markdown
- python3
- opencv
- cmake
- Eigen
```

#### terminal 

```shell
git clone [this code]
cd kevinzzx_Cpp

mkdir build && cd build

sudo cmake ..

sudo make 

```

#### compile

compile: cmake

IDE: vscode + clangd
